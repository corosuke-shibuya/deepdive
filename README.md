# 🔍 Deep Dive — AIコミュニケーション分析
> 会議ログをAIに分析させ、コミュニケーションスキルを可視化・改善するWebアプリ


## どんなプロダクトか
Deep Dive は、ビジネスパーソンの **会議コミュニケーション力を客観的に分析・改善** するためのツールです。

会議の発言録（テキストログ）を貼り付けるだけで、AIが以下を自動で行います。
- コミュニケーションの質を **0〜100点でスコアリング**
- 会話のズレ・意図のすれ違いを **具体的な場面つきで検出**
- 次回の会議に向けた **改善アクションとチェックリストを提案**
- 参加者ごとの **コミュニケーションタイプを分析・記録**

蓄積したデータから **自分のコミュニケーション傾向（マイプロフィール）** も自動生成されます。


## 主な機能

### 📊 ダッシュボード
- 過去の分析の平均スコアと推移をひと目で確認
- 直近5件の会議振り返り履歴を表示

### 🔄 会議振り返り（分析）
会議ログをAIが分析し、以下を出力します。

| 出力項目 | 内容 |
|---|---|
| 総合スコア | 0〜100点の円グラフ表示 |
| サブスコア | 明確さ・共感力・説得力・傾聴力・建設性・簡潔さの6軸 |
| コミュニケーション分析 | 場面・意図・相手の受け取り方・重要度をセットで提示 |
| 良い点（強み） | 会議で発揮できたポジティブな行動 |
| 改善アクション | 優先度付きの具体的な改善提案（言い換え例つき） |
| 次回チェックリスト | 次の会議前に確認すべき行動リスト |
| 参加者分析 | 参加者ごとのコミュニケーションタイプを分析|

### 🎯 会議準備
- **次の会議の目的・議題・懸念点** を入力
- 過去に分析した参加者プロフィールをもとに **コミュニケーション戦略を自動生成**
  - 会議の入り方・キーメッセージ・リスクと対処法・締め方
- 参加者情報がなくても、手動でメモを追加して準備可能

### 📁 準備履歴
- 過去に作成した会議準備の戦略を振り返り・再活用

### 🪞 マイプロフィール
- 複数回（2回以上）の分析データを集約して自動生成
- 自分のコミュニケーションタイプ・強み・盲点・成長エリアを可視化

## 使い方

### 1. アカウント登録・ログイン
アプリにアクセスしてメールアドレスとパスワードでアカウントを作成します。

### 2. 会議振り返り
1. サイドバーの「🔄 会議振り返り」をクリック
2. 会議ログ（テキスト形式）を貼り付ける
3. 必要に応じて「状況・背景」「参加者」「自分の呼称」を入力
4. 「分析する」をクリック → AIが約30〜60秒で分析を完了

> **ログのフォーマット例**
> ```
> 田中：では今日の議題に入ります。
> 鈴木：先週の件ですが、スケジュールの調整が難しくて…
> 田中：それは想定内です。では来週でどうでしょうか。
> ```
> 「名前：発言内容」の形式で書くと、参加者の自動検出精度が上がります。

### 3. 参加者を分析する
振り返り結果の「参加者」タブで「🔍 分析する」をクリックすると、その人の個別プロフィールが生成されます。

### 4. 会議準備
1. 「🎯 会議準備」をクリック
2. 過去に分析した参加者を選択（または手動でメモを追加）
3. 会議の目的・議題・懸念点を入力して「準備戦略を作成」

### 5. マイプロフィールを見る
分析を2回以上行った後、「🪞 マイプロフィール」→「プロフィールを生成」で自分の傾向を確認できます。
